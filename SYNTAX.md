# HEADING 
**Syntax :**  
```
# Heading 1  
## Heading 2  
### Heading 3  
#### Heading 4  
```
**Output :**    
# Heading 1  
## Heading 2  
### Heading 3  
#### Heading 4  

# TEXT FORMATTING
**Syntax :**  
```
**Bold** or __Bold__  
*Italic* or _Italic_  
~~Strikethrough~~   
**_Bold and Italic_**
```
**Output :**    
**Bold** or __Bold__  
*Italic* or _Italic_  
~~Strikethrough~~  
**_Bold and Italic_**
